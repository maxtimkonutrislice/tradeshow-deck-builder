# Nutrislice Tradeshow Deck Builder

An instant PowerPoint presentation generator for tradeshow events. Fill out a simple form and download a professional presentation in seconds.

## Features

✅ **Instant Generation** - Click "Generate PowerPoint" and download your PPTX immediately  
✅ **No Backend Required** - Runs entirely in your browser  
✅ **Offline Compatible** - Works without internet after loading  
✅ **Professional Output** - Clean, ready-to-present slides  
✅ **Customizable** - Edit the presentation after download in PowerPoint  

## How to Use

1. Open `tradeshow_instant_deck_builder.html` in any web browser
2. Fill in your event details across the five tabs:
   - Event Info
   - Attendees
   - Booth Details
   - Goals & Deadlines
   - Logistics
3. Click "Generate PowerPoint"
4. Your presentation downloads automatically

## Required Fields

- Event Name
- Location/City
- Start Date
- End Date
- Booth Number

All other fields are optional.

## What Gets Generated

The presentation includes:
- Title slide with event branding
- Show overview
- Team roster (if attendees added)
- Booth details & standard package
- Goals & lead targets
- Q&A closing slide

## Technical Details
